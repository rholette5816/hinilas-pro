# Task: Deploy to Vercel

## Goal
Deploy the current state of the web-hilas app to Vercel production.

## Steps
1. Make sure you are inside the `web-hilas/` directory
2. Run: `vercel --prod`
3. If prompted to confirm project settings, accept defaults
4. Report the production URL when done

## Do NOT
- Run `npm run build` manually before — Vercel handles the build
- Change any code
- Modify any config files

## Verification
- Deployment completes without errors
- Production URL is returned
